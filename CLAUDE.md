# Mobari MVP

Mobari is a Korean hair-loss management MVP.

## Goal

Build a fast MVP for:

- 탈모약 복용 체크
- 미녹시딜 체크
- 주간 사진 기록
- 전후 사진 비교
- 모발이 캐릭터 성장 UI

## Tech Stack

- Expo
- React Native
- TypeScript
- Expo Router

## Important Rules

- Keep code simple.
- Korean UI text.
- Do not build AI diagnosis yet.
- Do not build clinic matching yet.
- Avoid medical diagnosis wording.
- Use “변화 추적”, “관리 기록”, “참고용 분석” instead of “진단”.
